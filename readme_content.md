# Welcome to Axi

Axi is your personal AI assistant, living right here in Discord. A capable, autonomous AI that can hold conversations, run code, manage files, and orchestrate complex tasks — all from chat.

---

## How to Interact

**Main channel:** Type in **#axi-master** to talk to Axi directly — ask questions, give instructions, or just chat.

**Agent channels:** When Axi spawns an agent, it gets its own channel. Interact with that agent by typing in its channel.

---

## Agents

Axi can spawn **independent agent sessions** that work autonomously. Each gets its own Discord channel and runs in its own environment.

- *"Spawn an agent to [do X]"* — creates a new agent
- *"Kill [agent-name]"* — stops an agent and archives its channel
- `/list-agents` — shows all active agents

---

## Scheduling

Axi can schedule **recurring** or **one-off** events:

- *"Remind me to [X] at 3pm tomorrow"*
- *"Every weekday at 9am, ask me what I'm working on"*
- *"Cancel the morning standup reminder"*

---

## What You Can Ask

| What | Example |
|---|---|
| Spawn agent | *"Spawn an agent to refactor auth"* |
| Set reminder | *"Remind me at 5pm to check the deploy"* |
| Recurring task | *"Every Monday at 9am, summarize open PRs"* |
| Code tasks | *"What does the login function do?"* |
| Run commands | *"Run the test suite"* |
| Web search | *"Look up the latest Node.js LTS"* |

---

## Tips

- Axi remembers context within a session. Ask it to remember things across sessions — e.g. *"Always use bun instead of npm."*
- `/ping` to check if Axi is online. `/restart` if it seems stuck.
